what is HTTP?


The full form of HTTP is HyperText Transfer Protocol. It is designed for communication between web browsers and web servers. it is basically used for transmission of information across the internet. information is exchanged between clients and servers to and fro.



what is browsers and how they work?


it is a software program used to locate and display information on the internet. it is mostly used to access web pages. Browser send an HTTP request message to the server, asking it to send a copy of the website to the client.


What is Domain name?


Domain name is the text that a user enters into the browser window to reach a particular website. For eg. the domain name for google is google.com
